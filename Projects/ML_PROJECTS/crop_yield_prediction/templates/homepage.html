The aim here is to predict the quantity of blueberry crop yield using a regression machine learning model as this is a regression problem.

This project involves the display of the EDA performed on the data to gain insights and also the performance of the model during testing using RMSE as the performance metric.

A demonstration of the model predicting yield in real time is kept at last so that the model's predictions can be displayed according to data inserted by user.
